    50px / Normal / Serif


    Main uses:

    Most important header. (Use only when necessary)

    .lsn-xl{
    font-family: 'scala-reg';
    font-size: 50px;
    }


    .lsn-serif{
    font-family: 'scala-reg';
    font-size: 50px;
    }

    30px / Normal / Serif


    Main uses:

    Secondary Header / Callout for sections.

    .lsn-lg{
    font-family: 'scala-sans-reg';
    font-size: 30px;
    }

    .lsn-serif{
    font-family: 'scala-reg';
    font-size: 50px;
    }

    Main uses:

    Large Numbers ($ Total)

    18px / Normal / Sans-Serif


    Main uses:

    More important text (such as warnings or parts of the page the user must see)

    .lsn-lede{
    font-family: 'scala-sans-reg';
    font-size: 18px;
    }

    18px / Bold / Sans-Serif


    Main uses:

    Third level header. Most commonly used header on the page within sections.
    Button Text.

    .lsn-lede{
    font-family: 'scala-sans-reg';
    font-size: 18px;
    }

    .lsn-bold{
    font-family: 'scala-sans-bld';
    }

    15px / Normal / Sans-Serif


    Main uses:

    All normal text on the page.
    Footer, Popup, Paragraphs, Sentences, Form Fields...
    Used in Grid Columns.

    .lsn-body{
    font-family: 'scala-sans-reg';
    font-size: 15px;
    }

    15px / Bold / Sans-Serif


    Main uses:

    Used to emphasize normal body text or call out an important word or phrase.

    .lsn-body{
    font-family: 'scala-sans-reg';
    font-size: 15px;
    }

    .lsn-bold{
    font-family: 'scala-sans-bld';
    }

    13px / Normal / Sans-Serif


    Main uses:

    Header of tables. (when not selected to sort)
    Other small text labels.

    .lsn-sm{
    font-family: 'scala-sans-reg';
    font-size: 13px;
    }

    13px / Bold / Sans-Serif


    Main uses:

    Header of tables. (when selected to sort)
    Small and important text labels.

    .lsn-sm{
    font-family: 'scala-sans-reg';
    font-size: 13px;
    }

    .lsn-bold{
    font-family: 'scala-sans-bld';
    }
